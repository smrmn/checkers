# Шашки с модификаторами

Браузерная игра в шашки с дополнительными механиками. Один HTML-файл, без зависимостей.

## Запуск

Открыть `index.html` в браузере — больше ничего не нужно.

## Механики

**Классические шашки**
- Ход по диагонали вперёд, взятие — во всех 4 направлениях
- Обязательное взятие (шашки, которые должны бить, обводятся красным)
- Мульти-поедание: если после взятия есть ещё удары — ход продолжается
- Победа: все шашки противника съедены, либо больше шашек при заходе в тупик

**Хрупкие плитки** (toggle)
- Тёмные клетки разрушаются со временем: первый уход → трещина, второй → провал
- Провалившиеся клетки недоступны для хода и приземления
- Работает для обоих сторон

**Туман войны** (toggle)
- Видна только область в радиусе 2 клеток от каждой своей шашки
- Шашки бота в тумане не отображаются
- Активируется после первого хода игрока

## Бот

Приоритеты: самая длинная цепочка взятий → безопасные клетки → нетреснутые клетки → случайный ход. Задержка хода — 600 мс.

## Сборка из исходников

Исходники разбиты по файлам в `src/`. Сборка:

```bash
python3 build.py
```

Результат записывается в `index.html`.

```
src/
  template.html
  css/style.css
  js/01_state.js … js/09_game.js
```
